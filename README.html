# Mundo Nómada - Frontend E-commerce ## 📋 Descripción General Mundo Nómada es una plataforma e-commerce moderna y responsive desarrollada con Angular 19, que ofrece una experiencia de compra fluida para entusiastas de la moda. Esta aplicación frontend presenta una interfaz de usuario intuitiva, gestión robusta del carrito, autenticación segura de usuarios y capacidades completas de navegación por productos. ## ✨ Características Principales ### 🎨 Diseño Responsive Interfaz completamente adaptable optimizada para todos los dispositivos ### 🛍️ Catálogo de Productos Listados dinámicos de productos con filtrado por categorías y búsqueda ### 🛒 Carrito de Compras Avanzado - Verificación de stock en tiempo real - Sincronización automática del carrito entre usuarios invitados y registrados - Gestión inteligente de inventario con notificaciones al usuario - Cálculo detallado de precios (subtotal, IVA, gastos de envío) - Barra de progreso para envío gratuito ### 👤 Gestión de Cuentas de Usuario - Sistema de autenticación seguro - Personalización de perfil (nombre, email, contraseña) - Seguimiento del historial de pedidos ### 💳 Proceso de Checkout - Flujo de pago optimizado y amigable - Múltiples opciones de pago - Confirmación de pedido con generación de recibo ### ⚙️ Panel de Administración - Sistema completo de gestión de usuarios - Control de productos e inventario - Gestión de categorías ### 📝 Cumplimiento Legal - Sistema de consentimiento de cookies conforme al RGPD - Documentos legales completos (Política de Privacidad, Términos de Servicio) ## 🛠️ Tecnologías Utilizadas `Angular 19`, `TypeScript`, `Bootstrap 5`, `RxJS`, `HTML5`, `CSS3`, `LocalStorage`, `HttpClient`, `Reactive Forms`, `NgRx`, `HTML2Canvas`, `jsPDF`, `NGX-Toastr` ## 🚀 Instalación ```bash # Clonar el repositorio git clone https://github.com/tuusuario/mundo-nomada.git # Navegar al directorio del proyecto cd mundo-nomada # Instalar dependencias npm install # Iniciar servidor de desarrollo ng serve # Compilar para producción ng build --configuration production
